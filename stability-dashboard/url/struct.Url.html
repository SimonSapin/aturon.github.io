A Uniform Resource Locator (URL). A URL is a form of URI (Uniform Resource Identifier) that includes network location information, such as hostname or port number.

Example

extern crate url; fn main() { use url::Url; let raw = "https://username@example.com:8080/foo/bar?baz=qux#quz"; match Url::parse(raw) { Ok(u) => println!("Parsed '{}'", u), Err(e) => println!("Couldn't parse '{}': {}", raw, e), } }

Fields

scheme

The scheme part of a URL, such as https in the above example.

user

A URL subcomponent for user authentication. username in the above example.

host

A domain name or IP address. For example, example.com.

port

A TCP port number, for example 8080.

path

The path component of a URL, for example /foo/bar?baz=qux#quz.

Parses a URL, converting it from a string to a Url representation.

Arguments

  • rawurl - a string representing the full URL, including scheme.

Return value

Err(e) if the string did not represent a valid URL, where e is a String error message. Otherwise, Ok(u) where u is a Url struct representing the URL.
